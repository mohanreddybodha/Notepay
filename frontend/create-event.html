Need help creating?
A few tips to help you get started.
Use a clear event name
Choose a recognizable name for your event. e.g., "Tech Conference 2026" or "Alumni Meetup".
Keep the description short
A brief description gives participants context. You can always edit this later.
Goal amount is optional
Setting a goal adds a visual progress bar, which can help motivate contributors.
Powerful management features
After creation, you can add custom columns, invite collectors, and view detailed analytics.
Need help managing?
A few tips to help you manage your event effectively.
Adding Collectors
Collectors can join your event either by using the 16-character event code or by clicking your direct join link.
Public Read-Only View
If your event is set to Public, anyone with the public link can view event details in read-only mode.
Deactivate Event
Temporarily lock out collectors. They will not be able to join or access data until reactivated.
Delete Event
Permanently erase the event and all associated contributions and expenses.
