How to join an event?
Every event created on Notepay gets a unique 16-character invite code (e.g. A1B2C-3D4E-5F6G7).
Steps to follow:
1. Ask the Event Organizer to share their invite link or code.
2. Paste the exact code into the box below.
3. Click Join Event to instantly access the ledger.
